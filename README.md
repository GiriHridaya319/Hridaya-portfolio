A production-ready, highly animated portfolio for a full-stack AI/ML software engineer.

## Architecture

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Language**: TypeScript
- **Backend**: Next.js API Routes + Zod Validation
- **Theme**: Dark mode AI/ML aesthetic (Cyan & Purple accents)

## Key Features

- **Hero Section**: Dynamic typing animation with a neural-grid background.
- **Project Grid**: Interactive cards with tech tags and links.
- **Experience Timeline**: Professional journey with bullet-pointed achievements.
- **Skills Matrix**: Categorized technical expertise.
- **Contact Form**: API-powered with validation and status feedback.
- **SEO Optimized**: Metadata and OG tags configured for better reach.

## Setup & Running

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Deployment

This projects is optimized for **Vercel**. Simply push to a git repository and connect to Vercel for one-click deployment.
